Charcoal turns any photo into a portrait made entirely of keyboard characters. Upload a selfie, and the program scans it pixel by pixel, measures how bright each region is, and swaps every patch of the image for a character of matching "ink" — dense symbols like @ and % for shadows, airy ones like : and . for highlights. The result is your face redrawn as text art. You can pick between three art styles (classic shading, solid blocks, or a soft sketch look), choose how wide the piece should be, invert it for dark backgrounds, and save your favorite to a file. It's built on the image-processing ideas from Code in Place — reading pixels, calculating brightness, and nested loops — wrapped in a friendly little studio anyone can run on their own photos.
